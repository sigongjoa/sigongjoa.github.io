프로젝트 개요

AI-Hub 국가 데이터셋 구축

101.경상북도 어류 (북태평양 연어) 지능형 양식생산 데이터 구축 프로젝트는 AI-Hub 플랫폼을 통해 국가 차원의 양식업 AI 데이터셋을 구축하는 사업입니다.

구축된 다양한 데이터셋(수조 촬영 영상, 개체 촬영 실측, 수질 환경, 생육관리)에 대해서 AI 모델링을 진행하여 지능형 양식 시스템의 기반을 마련했습니다.

데이터셋 구성

영상 데이터

  • • 수조 촬영 영상 데이터: 연어 행동 패턴 분석
  • • 개체 촬영 실측 데이터: 개별 어류 형태학적 특성
  • • 다양한 각도와 조명 조건에서의 촬영
  • • 성장 단계별 어류 이미지 수집

환경 데이터

  • • 수질 환경 데이터: pH, 온도, 용존산소
  • • 생육관리 데이터: 사료 급여, 성장 기록
  • • 시간대별 환경 변화 모니터링
  • • 계절별 양식 환경 최적화 데이터

AI 모델링 구현

Segformer B5 기반 어류 분할

  • • Hugging Face Transformers: 사전 훈련된 Segformer B5 모델 활용
  • • 수조 영상 분석: 개별 어류 자동 식별 및 분할
  • • 실시간 처리: 양식장 모니터링 시스템 연동
  • • 정확도 최적화: 수중 환경 특화 모델 튜닝

LightGBM 기반 성장 예측

  • • 그래디언트 부스팅: 환경 데이터 기반 성장률 예측
  • • 특성 중요도 분석: 어류 성장에 미치는 주요 환경 요인 식별
  • • 최적화된 성능: 빠른 학습과 높은 예측 정확도
  • • 실무 적용: 양식 관리 의사결정 지원

TFT(Temporal Fusion Transformer) 시계열 예측

  • • PyTorch-Forecasting: 시계열 데이터 전용 트랜스포머
  • • 멀티 호라이즌 예측: 단기/중기/장기 성장 전망
  • • 어텐션 메커니즘: 시간별 중요도 자동 학습
  • • 해석가능성: 예측 근거 시각화 제공

기술적 혁신점

통합 AI 플랫폼

컴퓨터 비전, 머신러닝, 시계열 예측을 통합한 종합적인 지능형 양식 시스템을 구축했습니다. 각각의 모델이 독립적으로 작동하면서도 상호 보완적인 정보를 제공하여 양식업의 전 과정을 AI로 최적화할 수 있는 기반을 마련했습니다.
