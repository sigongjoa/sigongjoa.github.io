프로젝트 개요

면접 평가 시스템의 디지털 혁신

기존 종이 기반 면접 평가의 비효율성을 해결하기 위해 Windows Desktop과 Android 태블릿을 연동한 크로스 플랫폼 시스템을 개발했습니다.

면접관은 태블릿으로 직관적인 점수 입력을, 관리자는 PC에서 통합 관리를 할 수 있는 효율적인 솔루션입니다.

핵심 문제 해결

기존 문제점

  • • 종이 기반 평가의 비효율성
  • • 수기 점수 집계의 오류 가능성
  • • 면접 자료 관리의 복잡성
  • • 실시간 진행 상황 파악 어려움

솔루션 제공

  • • 태블릿 기반 직관적 점수 입력
  • • 자동 데이터 집계 및 CSV 출력
  • • PDF 자동 업로드 및 분류
  • • 실시간 진행 모니터링

시스템 아키텍처

크로스 플랫폼 설계

🖥️ Windows Desktop (관리자)

  • • 면접 정보 설정 및 PDF 업로드
  • • 지원자 그룹 관리 (드래그 앤 드롭)
  • • ADB를 통한 설정 파일 전송
  • • 결과 수집 및 CSV 내보내기

📱 Android Tablet (면접관)

  • • 직관적인 점수 입력 인터페이스
  • • PDF 뷰어 연동 지원자 확인
  • • 실시간 데이터 유효성 검증
  • • 평가 완료 시 JSON 자동 저장

ADB 연동 기술

📋 핵심 ADB 명령어

// 기기 연결 확인
adb devices

// 설정 파일 전송
adb push config.json /sdcard/Download/interPad/interview_ready/

// 결과 파일 다운로드
adb pull /sdcard/Download/interPad/interview_done/ ./results/

// MTP 모드 확인
adb shell getprop sys.usb.config
