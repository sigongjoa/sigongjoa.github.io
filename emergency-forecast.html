프로젝트 개요

시공간 데이터 기반 응급구조 수요 예측

강원도 소방본부의 응급구조 데이터와 공공 데이터를 결합하여 지능형 응급수요 예측 시스템을 개발했습니다.

Temporal Fusion Transformer(TFT) 모델을 활용하여 시공간적 패턴을 학습하고 교통사고 위험을 예측하는 시스템을 구축했습니다.

기술적 구현

데이터 수집 및 전처리

  • • 강원도 소방일지 데이터: 응급 출동 기록, 사고 유형, 발생 시간/장소
  • • 공공 데이터: 교통량, 날씨, 지리적 정보, 인구 통계
  • • 시공간 특성 추출: 시간대별, 요일별, 계절별 패턴 분석
  • • 지리적 특성: 도로 유형, 교차로 밀도, 지역 특성 정보

Temporal Fusion Transformer 적용

  • • 멀티헤드 어텐션: 시공간적 의존성 모델링
  • • 변수 선택 네트워크: 중요한 특성 자동 선별
  • • 시계열 예측: 다양한 시간 호라이즌에서의 수요 예측
  • • 해석가능성: 어텐션 가중치를 통한 예측 근거 제공

모델 최적화

  • • 하이퍼파라미터 튜닝: Optuna를 활용한 자동 최적화
  • • 데이터 증강: 시계열 데이터 특성을 고려한 증강 기법
  • • 교차 검증: 시간 기반 분할을 통한 견고한 평가
  • • 앙상블 기법: 여러 모델의 예측 결합으로 성능 향상

프로젝트 임팩트

실무 적용
